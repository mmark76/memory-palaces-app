Memory Palace

Usage Instructions

1. Creating a Palace: Type a name and press "Create".

2. Selecting a Palace: Click on a palace to open it.

3. Adding a Locus: Fill in on the left the description of the locus (place) and on the right what you want to remember, then press "Add".

4. Viewing Loci: Each row in the list has two columns: on the left the locus, on the right the association/memory you have defined.

5. Note: Data is not stored permanently without using localStorage (if you close the page it is lost).
