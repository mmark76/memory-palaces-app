Memory Palaces Application

Example memory palaces

These examples show how different types of information fit into the same memory palace (the same house with 10 loci). Each example uses the same places, but with different things to remember: shopping, recipes, rivers, countries, history, geography, studies. You can read the examples, copy the JSON, or use them as a template to build your own memory palaces.

Select an example to preview its JSON memory palace. 


      

    



    
    


    

      
Usage Instructions


      
1. Creating a Palace: Type a name and press "Create".

      
2. Selecting a Palace: Click on a palace to open it.

      
3. Adding a Locus: Fill in on the left the
        description of the locus (place) and on the right what
        you want to remember, then press "Add".

      
4. Viewing Loci: Each row in the list has two columns:
        on the left the locus, on the right the association/memory you have defined.

      
5. Note: Data is not stored permanently without using localStorage
        (if you close the page it is lost).
